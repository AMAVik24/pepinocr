A simple but powerful plugin to set up a business website.

== Description ==

This plugin sets up essential features for a simple business website in WordPress 6.2.2. Features include email SMTP configuration, SEO enhancements like Meta Description and Schema Markup, security measures like maximum login attempts, and child theme configuration for enqueue. It's lightweight and translation-ready.

== Installation ==

1. Upload `ama-site-essentials.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin settings through the provided options in the WordPress dashboard

== Frequently Asked Questions ==

= How do I configure email SMTP settings? =

You can configure email SMTP settings under the plugin settings in the WordPress dashboard. Enter your SMTP server details and authentication credentials there.

= How can I optimize my website for SEO using this plugin? =

This plugin provides features like Meta Description and Schema Markup settings to enhance your website's SEO. You can configure these settings under the plugin options.
